About This Project

Book & Author Hub is a web app built as a final project for WDD 330. It allows users to search books using the Open Library API and fetch detailed data and cover images from the Google Books API.

You can view book details, mark favorites, copy them to your clipboard, and download your favorite list as text or JSON files.
